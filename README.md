## Why SillyChar?

Making a good card is hard. Making a *world* is harder. You've read the guides, stared at blank fields, and it still feels overwhelming. **SillyChar-Creator V2** fixes that. Paste it into ChatGPT, Claude, Gemini, or any capable AI, describe what you want, and get a complete build back.

No configuration. No setup. Just paste and describe.

## in only 6K Token!

### What You Get

**Characters:**
- **Novel-quality first messages** — 7-12 paragraphs with atmosphere, sensory detail, and a character already *doing something*
- **Distinct voices** — speech profiles with filler words, vocabulary range, and sentence structure so every character *sounds* different
- **Structured bracket format** — clean and consistent:`Appearance: face{...},body{...}`,`Relationships[ {{user}}: ... ]`
- **4-8 alternate greetings** — each a full scene with different moods, timelines, and stakes
- **Danbooru tag generation** — image-ready tags pulled from your character's appearance

**Worlds:**
- **Full world cards** — narrator persona with embedded lorebook, faction anchors, dynamic entry loading
- **Atmospheric narrator hooks** — scene-setting opens that establish tone, texture, and mood
- **Multiple entry points** — different eras, locations, or social positions to start from

**Lorebooks:**
- **Standalone World Info JSON** — import directly to the World Info panel
- **Recursive tree architecture** — content loads dynamically as it becomes relevant
- **Clean organization** — characters, locations, factions, items, events, rules

**And More:**
- **Persona creation** — your roleplay avatar with optional lorebook
- **Group chat builds** — multiple character cards with shared lorebook
- **Soul Extractor** — paste text, get a reusable style guide to preserve any voice
- **Built-in lorebook updates** — add to existing lorebooks without collisions
- **Large cast support** — automatic tier handling for 25+ characters

## Six Modes

| Mode | What It Builds |
|------|----------------|
| **Character** | Single, import-ready character card |
| **World** | World card with narrator persona and embedded lorebook |
| **Lorebook** | Standalone World Info JSON |
| **Persona** | User avatar with optional lorebook |
| **Group Chat** | Multiple character cards + shared lorebook |
| **Soul Extractor** | Reusable prose style guide from any text |

### Build Options

Stack with any mode:

| Option | Effect |
|--------|--------|
|`quick`| Describe → generate directly |
|`guided`| Step-by-step with checkpoints |
|`brainstorm`| Develop concept first, then build |
|`skip research`| No inventory phase |
|`full detail`| No token ceilings |
|`add world state`| Reactive world state block |

## Works Everywhere

SillyChar generates standard **SillyTavern V2 character cards** and **World Info JSON**. Import into:

- **[RoleCall](https://rolecallstudios.com/)** — cards include RoleCall extension data (colors, pronouns, metadata) out of the box
- **SillyTavern** — drop the JSON in, done
- **Any SillyTavern-compatible platform**

### A Note on RoleCall

RoleCall is a great place to share and play cards — but it can feel *premium*. Polished. Like your first card won't belong there.

**It will.** You don't need to be a veteran card maker. You don't need to know JSON. Describe what you want, get your build, import it.

**After import:** Tag your card properly on RoleCall. Genre, archetype, setting, personality — SillyChar generates tag suggestions for you, so you're not guessing.

## Quick Start

1. Download **[SillyChar-Creator.md](https://github.com/artixr1/SillyChar-Creator/releases/tag/v2.0)**
2. Paste into your AI of choice (ChatGPT, Claude, Gemini, NanoGPT, etc.)
3. Pick a mode: **Character** / **World** / **Lorebook** / **Persona** / **Group Chat** / **Soul Extractor**
4. Describe what you want
5. Import the JSON — SillyTavern, RoleCall, or anywhere
6. Tag it properly — done
